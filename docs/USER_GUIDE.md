# User Guide

Complete usage guide for MS KMS Activator v3.5.0

## Quick Start

1. Download `KMS_Activator.exe` from [Releases](https://github.com/50bvd/kmsact/releases)
2. **Right-click** → **Run as Administrator**
3. Click **Activate Windows** or **Activate Office**

## Features

### Windows Activation
- Supports Windows 10/11 Pro, Education, Enterprise
- Home edition must be upgraded first (use Edition Changer)
- Activation valid for 180 days (use Auto-Renewal)

### Office Activation  
- Supports Office 2016/2019/2021/2024 LTSC (Volume)
- Not for Office 365/Microsoft 365
- Activates all installed Office products

### Edition Upgrade
- Upgrade Home → Pro → Education/Enterprise
- Requires system restart
- Save all work before starting

### Office Installer
- Downloads Office LTSC 2024 from Microsoft
- Choose: ProPlus, Visio, Project
- Takes 15-30 minutes to install

## Settings

- **Language**: English, French, Spanish, German, Italian (auto-detected)
- **Theme**: Light/Dark (auto-detected from Windows)
- **KMS Server**: Customizable (default: kms.50bvd.com)

## Troubleshooting

**Access Denied** → Run as Administrator  
**Home edition not supported** → Use Edition Changer first  
**Office not activating** → Install Volume edition using Office Installer  
**Installation hangs** → Wait patiently, takes 15-30 minutes

For detailed help, see full documentation or [open an issue](https://github.com/50bvd/kmsact/issues).
